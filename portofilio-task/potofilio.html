Here a description of a few projects I did:

A University Class Registration System allows students to log in, view available courses, and register for classes. It supports real-time updates on class availability and waiting lists. Students can search for courses and plan their semester based on degree requirements. Administrators manage course offerings and student enrollment securely through the platform.

A Food Chain Ordering App UI/UX focuses on delivering a seamless, intuitive experience for users to browse menus, customize orders, and make secure payments. The interface features easy navigation, clear visuals, and personalized suggestions. Users can track orders in real-time and access past orders for quick reordering. The design prioritizes accessibility, speed, and user convenience across devices.

A To-Do List Website with a Fun Gaming Theme blends task management with a playful, interactive design inspired by video games. Users can track tasks, set goals, and earn rewards or badges for completing them. The UI features dynamic, game-like elements such as progress bars, levels, and achievements. The fun theme keeps users motivated while providing an efficient way to organize and prioritize their tasks.

A Game-Inspired To-Do List Website turns tasks into in-game quests, with users unlocking levels as they complete them. The UI features themed graphics, icons, and sound effects for an immersive experience, making productivity feel like an adventure while keeping task management efficient and fun.
